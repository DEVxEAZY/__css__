Fundamentos de CSS

Um único índice: cada link abre o exemplo prático na pasta correspondente e, abaixo, o que esse tópico cobre em uma frase.

  1. Passo 1 Conceito de CSS e estilo inline

    O CSS define aparência; o estilo inline vai direto na tag (style=""), útil para testes rápidos, pouco recomendável em escala.

  2. Passo 2 CSS interno (na página)

    Regras dentro de <style> no <head>: já separa um pouco do HTML, mas vale para uma página só ou protótipos.

  3. Passo 3 CSS externo (arquivo .css)

    Um arquivo .css ligado com <link>: reutilização entre páginas, cache do navegador e manutenção típica em projetos reais.

  4. Passo 4 Seletores e classes

    Seletores escolhem quais elementos ganham cada regra; classes (.nome) agrupam estilos reutilizáveis sem repetir marcação específica demais.

  5. Passo 5 Propriedade color

    Define a cor do texto (e, em geral, do conteúdo “currentColor” de ícones/SVG associados). Base para contraste e hierarquia visual.

  6. Passo 6 Propriedade background

    Cor de fundo, imagens, posição e repetição atrás do conteúdo; compõe atmosfera da página sem alterar estrutura do HTML.

  7. Passo 7 Propriedade border

    Contorno ao redor da caixa: espessura, estilo e cor; também influencia radii (border-radius) para cantos arredondados.

  8. Passo 8 Propriedade margin

    Espaço externo entre caixes (afasta um bloco dos vizinhos). Colapso de margens é um detalhe importante entre blocos adjacentes.

  9. Passo 9 Propriedade padding

    Espaço interno entre a borda e o conteúdo; aumenta a área clicável e respiração visual sem mover outros elementos tanto quanto margem externa.

  10. Passo 10 width e height

    Define tamanhos explícitos da caixa de conteúdo (combinando com modelo de caixa para bordas e padding).

  11. Passo 11 Propriedades de texto

    Alinhamento, espaçamento entre letras e linhas, decoração (text-decoration), indentação etc.: tipografia dentro do fluxo.

  12. Passo 12 Fontes (font-*)

    Família, tamanho, peso (font-weight), estilo italic; combinado ao texto forma a identidade legível da interface.

  13. Passo 13 Estilizando links

    Pseudoclasses (:link, :visited, :hover, :focus) e estados acessíveis; links devem parecer clicáveis e ter foco visível.

  14. Passo 14 Propriedade display

    Define o modo de layout: bloco, inline, flex, grid, none… é a chave para flexbox (display: flex) e grid (display: grid).
